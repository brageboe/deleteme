**First things first - there are 2 ways of installing mods: CKAN \& Manual Install.**

\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_





CKAN is like an auto-installer mod database. It's not a mod, but a second program that will install mods for you. It is a great tool that will automate most of the updates \& dependency checks. Download \& install CKAN here if you haven't already: [https://forum.kerbalspaceprogram.com/topic/90246-the-comprehensive-kerbal-archive-network-ckan-package-manager-v1180-19-june-2016/](https://forum.kerbalspaceprogram.com/topic/90246-the-comprehensive-kerbal-archive-network-ckan-package-manager-v1180-19-june-2016/)



The best part is you can use it to automatically install like 95 percent of the mods I have using the CKAN MODPACK file that is included in the folder.



Download the **.ckan** file in this folder called **Bogue's 2026 Modpack** - which is just basically like a text list that tells CKAN what mods to add.



Click on **File** in the top left corner, then click **import from .ckan**. It will open your file explorer, where you should select **Bogue's 2026 Modpack.ckan** and hit **open**. This will import my modlist into CKAN - where you can hit the **Apply** button next to the green check mark and it will install the whole mod list.



Make sure under CKAN **Settings** - right next to File -, and under Compatible Game Versions, you check all boxes for the latest game version down to like **1.8**. This will double check that CKAN will allow mods that may not have been updated for a while to still be installed.



And boom. That's like 95 percent of my list installed already.



If you need a tutorial on how to install CKAN - check out the great tutorials that are on Youtube. Here's a good one by KSP legend VAOS: [https://www.youtube.com/watch?v=Fbe18MMK6wk](https://www.youtube.com/watch?v=Fbe18MMK6wk)

**Volumetric Clouds V5** is a mod that costs a few dollars on the legendary Modder Blackrack's patreon page. Its well worth paying for to download, you can even download it and cancel the subscription right after. Download and install instructions found here: [https://www.patreon.com/posts/139881534](https://www.patreon.com/posts/139881534)

**Promised Worlds** - KSP 2's cancelled Interstellar worlds brought to KSP1



* available on CKAN
* Works with Parallax Continued \& Volumetric Clouds automatically
* type Promised Worlds in the search bar on CKAN, select Promised Worlds - Core, and the systems you would like to add, then hit apply to install.



**Kerbal Star Systems 2** - Huge modular interstellar planet mod with a bunch of star systems that you can choose from or add all at once



* Manual install required. Download here:  [https://github.com/StarCrusher96/Kerbal-Star-Systems-2/releases](https://github.com/StarCrusher96/Kerbal-Star-Systems-2/releases)
* Install instructions here:  [https://github.com/StarCrusher96/Kerbal-Star-Systems-2/wiki/How-to-Install-Kerbal-Star-Systems-2](https://github.com/StarCrusher96/Kerbal-Star-Systems-2/wiki/How-to-Install-Kerbal-Star-Systems-2)
* Works with Parallax Continued \& Volumetric Clouds automatically
* Install tutorial video: [https://www.youtube.com/watch?v=bGw612trMy4](https://www.youtube.com/watch?v=bGw612trMy4)
* Make sure to unzip / extract the KSS2 download files. Your browser will most likely compress them because of their larger size, which will make them not work when installed if not unzipped / extracted first. Make sure to unzip / extract the download files to a new location and use those uncompressed files to install.
* Has replacement to Outer Planets Mod \& Minor Planets Expansion in development, which is good because OPM \& MPE are getting kinda old.

 

**Cosmic Serenity** - Interstellar planet mod with exotic star systems from a pink brown dwarf to a blue pulsar, each with their own planets!



* available on CKAN
* Works with Parallax Continued \& Volumetric Clouds automatically
* type Promised Worlds in the search bar, select Cosmic Serenity - Core, and the systems you would like to add, then hit apply to install.





**IMPORTANT notes for planet packs**

\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_



Having a ton of planet packs installed all at once can increase the game loading times and may slow down performance. **I recommend only having 1 or 2 interstellar planet packs installed at once**. If you have 3 installed at once, you probably will need a really good PC to run that without lag.



Also note that in-game, it will probably take longer than usual for the game to load modded planets in for the first time. Be patient when loading in a planet pack for the first time so that the game can get familiar with the new loading paths and it will load the planets faster in the future.
